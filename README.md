Desafio API - To Do List

API REST de gerenciamento de tarefas desenvolvida com Spring Boot.

- Tecnologias utilizadas

- Java 17+
- Spring Boot
- Spring Web
- Spring Data JPA
- Maven
- Banco de dados H2 / PostgreSQL 

---

- :clipboard: Funcionalidades

- :white_check_mark: Criar tarefa
- :page_facing_up: Listar tarefas
- :pencil2: Atualizar tarefa
- :x: Deletar tarefa
